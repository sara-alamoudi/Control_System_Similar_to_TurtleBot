# Control_System_Similar_to_TurtleBot
The system was created using TinckerCad

The components of the circuit were as following:

*Arduino Uno

*UltraSonic Distance Sensor

*Battery

-Two DC Motors
-LCD 
-Breadboard Small
-Breadboard Mini
-Potentiometer(Control the lighting of the LCD)
-Resistor
